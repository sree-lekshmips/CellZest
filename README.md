# CellZest

An Anemia Prediction and Food Recommendation System developed using Django and Machine Learning.

## Features
- Predict anemia severity
- Recommend foods based on hemoglobin level
- Generate personalized meal plans
- User-friendly web interface

## Technologies Used
- Python
- Django
- Pandas
- Scikit-learn
- Matplotlib

## Installation

```bash
pip install -r requirements.txt
python manage.py runserver
```
